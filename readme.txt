Use Radical to block spam comments

== Description ==

Radical just takes the comments that contain an URL to the spam folder.

== Installation ==

1. Go to Plugins menu > Add new
2. Search for "Radical"
3. Activate the plugin through the ‘Plugins’ menu in WordPress

== Frequently Asked Questions ==

= How works this plugin? =
Radical just takes the comments that contain an URL to the spam folder.

= Could a real comment go to the spam folder? =
Yes, use Radical only if your users usually comment without using an URL
